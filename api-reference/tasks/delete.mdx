---
title: 'Delete task'
description: 'Permanently delete a task.'
---

`DELETE /api/tasks/{task_id}`

Permanently deletes a task. Regular users can only delete their own tasks. Admin users can delete any task.

<Info>**Authentication:** Bearer access token required.</Info>

<Warning>This action is permanent and cannot be undone.</Warning>

## Path parameters

<ParamField path="task_id" type="integer" required>
  The ID of the task to delete.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean">
  Whether the request succeeded.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable result message.
</ResponseField>

<RequestExample>

```bash cURL

  --url https://api.taskforge.example.com/api/tasks/1 \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```
