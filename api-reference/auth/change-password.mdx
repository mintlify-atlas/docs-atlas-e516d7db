---
title: 'Change password'
description: 'Change the password for the currently authenticated user.'
---

`POST /api/auth/change-password`

Updates the password for the current user. On success, all existing refresh tokens for the account are revoked for security.

<Info>**Authentication:** Bearer access token required.</Info>

<Info>**Rate limit:** 3 requests per hour per IP address.</Info>

## Request headers

<ParamField header="Authorization" type="string" required>
  Bearer access token. Format: `Bearer <access_token>`
</ParamField>

## Request body

<ParamField body="old_password" type="string" required>
  The user's current password.
</ParamField>

<ParamField body="new_password" type="string" required>
  The desired new password. Must be 8–128 characters and contain at least one uppercase letter, one lowercase letter, and one number.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean">
  Whether the request succeeded.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable result message.
</ResponseField>

<Warning>
  Changing your password revokes all active refresh tokens. You will need to log in again on all other devices.
</Warning>

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.taskforge.example.com/api/auth/change-password \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...' \
  --header 'Content-Type: application/json' \
  --data '{
  "old_password": "OldPass123",
  "new_password": "NewPass456"
}'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "success": true,
  "message": "Contrasena cambiada con exito"
}
```

```json 400
{
  "success": false,
  "error": "La contrasena debe contener al menos una letra mayuscula"
}
```

```json 401
{
  "success": false,
  "error": "La contrasena actual es incorrecta"
}
```

</ResponseExample>
