---
title: 'Create tag'
description: 'Create a new tag for categorizing tasks in TaskForge.'
---

`POST /api/tags`

Creates a new tag. Tag names must be unique across all tags.

<Info>
  Requires a valid Bearer access token. All authenticated users can create tags.
</Info>

<Note>
  This endpoint is rate-limited to **20 requests per hour** per user.
</Note>

## Body parameters

<ParamField body="name" type="string" required>
  Display name for the tag. Must be between 1 and 50 characters. Must be unique.
</ParamField>

<ParamField body="color" type="string" default="#808080">
  Hex color code for the tag. Must match the pattern `#RRGGBB` (e.g., `#FF5733`). Defaults to `#808080` if not provided.
</ParamField>

<ParamField body="description" type="string">
  Optional description of the tag. Maximum 255 characters.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  Whether the request succeeded.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

<ResponseField name="data" type="object" required>
  The newly created tag object.
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Unique identifier for the tag.
    </ResponseField>
    <ResponseField name="name" type="string">
      Tag display name.
    </ResponseField>
    <ResponseField name="color" type="string">
      Hex color code for the tag.
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description of the tag.
    </ResponseField>
    <ResponseField name="task_count" type="integer">
      Number of tasks associated with this tag. Will be `0` for a newly created tag.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp when the tag was created.
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp when the tag was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.taskforge.com/api/tags' \
  --header 'Authorization: Bearer <access_token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Work",
    "color": "#FF5733",
    "description": "Work-related tasks"
  }'
```

</RequestExample>

  "success": true,
