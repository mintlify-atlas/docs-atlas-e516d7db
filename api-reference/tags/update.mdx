---
title: 'Update tag'
description: 'Update an existing tag by its ID.'
---

`PUT /api/tags/{tag_id}`

Updates one or more fields on an existing tag. All body fields are optional — only fields provided in the request will be updated. Also accepts `PATCH`.

<Note>
  **Admin only.** This endpoint requires the `admin` role.
</Note>

## Path parameters

<ParamField path="tag_id" type="integer" required>
  The unique identifier of the tag to update.
</ParamField>

## Body parameters

<ParamField body="name" type="string">
  New display name for the tag. Must be between 1 and 50 characters. Must be unique.
</ParamField>

<ParamField body="color" type="string">
  New hex color code for the tag. Must match the pattern `#RRGGBB` (e.g., `#FF5733`).
</ParamField>

<ParamField body="description" type="string">
  New description for the tag. Maximum 255 characters. Pass `null` to clear the description.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  Whether the request succeeded.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message.
</ResponseField>

<ResponseField name="data" type="object" required>
  The updated tag object.
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Unique identifier for the tag.
    </ResponseField>
    <ResponseField name="name" type="string">
      Tag display name.
    </ResponseField>
    <ResponseField name="color" type="string">
      Hex color code for the tag.
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description of the tag.
    </ResponseField>
    <ResponseField name="task_count" type="integer">
      Number of tasks currently associated with this tag.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp when the tag was created.
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp when the tag was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request PUT \
  --url 'https://api.taskforge.com/api/tags/1' \
  --header 'Authorization: Bearer <access_token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Work",
    "color": "#FF5733",
    "description": "Work-related tasks"
  }'
```

</RequestExample>
